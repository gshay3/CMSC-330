📘 CMSC-330 Projects Collection

This repository contains programming assignments completed for CMSC 330 (Organization of Programming Languages). The projects explore object-oriented design, graphics rendering, and expression evaluation across multiple programming languages including Java and C++.

Each project demonstrates core concepts such as inheritance, polymorphism, geometric computation, and abstract syntax tree evaluation.
 <br> <br> <br>
📂 Projects Overview

🔹 Project 1: Java Graphics & Polygon System A Java-based graphics system that models and renders geometric shapes using object-oriented design.

Features:

Implements multiple polygon types (Isosceles Triangle, Parallelogram, Regular Polygon) <br>
Dynamically computes vertex coordinates for shapes <br>
Uses trigonometric calculations for regular polygons <br>
Supports filled polygon rendering using Graphics API <br>
Includes text rendering within a graphical scene <br>
Utilizes inheritance through base classes (Polygon_, SolidPolygon, Image) <br>

Concepts Covered:

Object-oriented programming (inheritance and polymorphism) <br>
Java AWT graphics rendering <br>
Coordinate geometry and transformations <br>
Trigonometric calculations (sine and cosine) <br>
Abstract class design and extension <br>
 <br> <br>
🔹 Project 2: Expression Evaluation System (C++) A C++-based system that evaluates mathematical expressions using an object-oriented hierarchy of expression classes.

Features:

Implements arithmetic operations as classes (Division, Multiplication, Power, Remainder) <br>
Supports comparison-based operations (Min, Max) <br>
Uses a binary expression tree structure <br>
Recursively evaluates expressions through function calls <br>
Encapsulates operations using a common SubExpression base class <br>

Concepts Covered:

Object-oriented programming in C++ <br>
Expression trees and recursive evaluation <br>
Polymorphism and dynamic dispatch <br>
Mathematical computation and operator abstraction <br>
Class hierarchies and inheritance <br>
